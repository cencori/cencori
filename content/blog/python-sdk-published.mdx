Today we published the official Cencori Python SDK to PyPI.

## Installation

```bash
pip install cencori
```

## Quick Start

```python
from cencori import Cencori

cencori = Cencori(api_key="your-api-key")

response = cencori.ai.chat(
    messages=[{"role": "user", "content": "Hello!"}],
    model="gpt-4o",
)

print(response.content)
```

## Streaming

```python
for chunk in cencori.ai.chat_stream(
    messages=[{"role": "user", "content": "Tell me a story"}],
    model="claude-3-opus",
):
    print(chunk.delta, end="")
```

## What's Included

- **Full feature parity** with TypeScript SDK
- **Streaming support** via httpx
- **Type-safe dataclasses** for all responses
- **Comprehensive error handling** with typed exceptions

Published to [pypi.org/project/cencori](https://pypi.org/project/cencori/).
