Generate images from text prompts using OpenAI's GPT Image, DALL-E, or Google's Imagen models.

## Basic Request

```typescript
const response = await cencori.ai.generateImage({
  prompt: 'A futuristic city at sunset, cyberpunk style',
  model: 'gpt-image-1.5',
  size: '1024x1024'
});

console.log(response.images[0].url);
```

## Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `prompt` | string | Yes | Image description |
| `model` | string | Yes | Model identifier |
| `size` | string | No | Image dimensions |
| `quality` | string | No | 'standard' or 'hd' |
| `n` | number | No | Number of images |
| `style` | string | No | 'natural' or 'vivid' |

## Response

```typescript
{
  images: [
    {
      url: 'https://...',
      revisedPrompt: 'A sprawling futuristic city...'
    }
  ],
  model: 'gpt-image-1.5'
}
```

## Available Sizes

| Model | Sizes |
|-------|-------|
| gpt-image-1.5 | 1024x1024, 1792x1024, 1024x1792 |
| dall-e-3 | 1024x1024, 1792x1024, 1024x1792 |
| dall-e-2 | 256x256, 512x512, 1024x1024 |
| imagen-3 | 1024x1024 |

## HTTP API

```bash
curl -X POST https://cencori.com/api/ai/images/generate \
  -H "CENCORI_API_KEY: csk_..." \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A futuristic city at sunset",
    "model": "gpt-image-1.5",
    "size": "1024x1024"
  }'
```

## Supported Models

| Provider | Models |
|----------|--------|
| OpenAI | gpt-image-1.5, gpt-image-1, dall-e-3, dall-e-2 |
| Google | gemini-3-pro-image, imagen-3 |
