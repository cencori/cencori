## Installation

```bash
composer require cencori/cencori-php
```

## Basic Usage

```php
<?php

require_once 'vendor/autoload.php';

use Cencori\Cencori;

$cencori = new Cencori(['apiKey' => getenv('CENCORI_API_KEY')]);

// Chat
$response = $cencori->ai->chat(
    messages: [['role' => 'user', 'content' => 'Hello!']],
    model: 'gpt-4o',
);

echo $response->content;
```

## Sessions

```php
// Create a session
$session = $cencori->sessions->create([
    'agent_id' => 'ag_...',
]);

// Submit a turn
$turn = $cencori->sessions->submitTurn($session['id'], [
    'input' => 'What is the weather?',
]);

// Approve a pending action
$cencori->sessions->approve($session['id'], [
    'action_id' => 'act_...',
    'tool_results' => [
        ['action_id' => 'act_...', 'output' => 'Sunny, 72°F'],
    ],
]);
```

## Agents

```php
// Create an agent
$agent = $cencori->agents->create([
    'name' => 'my-agent',
    'config' => [
        'model' => 'gpt-4o',
        'system_prompt' => 'You are a helpful assistant.',
    ],
]);
```

## Memory

```php
// Store a memory
$cencori->memory->store([
    'namespace' => 'docs',
    'content' => 'Refund policy allows returns within 30 days',
]);

// Semantic search
$results = $cencori->memory->search([
    'namespace' => 'docs',
    'query' => 'what is our refund policy?',
    'limit' => 5,
]);
```

## Configuration

```php
$cencori = new Cencori([
    'apiKey' => 'csk_...',
    'baseUrl' => 'https://api.cencori.com',
    'timeout' => 30.0,
]);
```
