## BYOK (Bring Your Own Keys)

Use your own API keys for direct billing with providers:

```typescript
const cencori = new Cencori({
  apiKey: 'csk_...',
  providers: {
    openai: { apiKey: 'sk-...' },
    anthropic: { apiKey: 'sk-ant-...' },
    google: { apiKey: 'AIza...' },
  }
});
```
