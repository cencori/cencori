Cencori works seamlessly in serverless environments like Firebase Cloud Functions and Supabase Edge Functions.

## Firebase Integration

Use the Cencori Node.js SDK within your Cloud Functions.

```typescript
// functions/src/index.ts
import * as functions from "firebase-functions";
import { Cencori } from "cencori";

const cencori = new Cencori({
  apiKey: process.env.CENCORI_API_KEY,
});

export const chatWithAI = functions.https.onCall(async (data, context) => {
  // 1. Validate auth
  if (!context.auth) throw new functions.https.HttpsError("unauthenticated", "Login required");

  // 2. Call Cencori
  const response = await cencori.ai.chat({
    model: "gpt-4o",
    messages: [{ role: "user", content: data.message }],
    user: context.auth.uid // Pass User ID for analytics
  });

  return { response: response.content };
});
```

## Supabase Integration

### Edge Functions

Supabase Edge Functions run on Deno. You can use `fetch` or the Cencori SDK (if ESM compatible).

```typescript
// supabase/functions/ai-chat/index.ts
import { serve } from "https://deno.land/std@0.168.0/http/server.ts"

serve(async (req) => {
  const { query } = await req.json()

  const response = await fetch('https://cencori.com/api/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${Deno.env.get('CENCORI_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      model: 'gpt-4o',
      messages: [{ role: 'user', content: query }],
    }),
  })

  const data = await response.json()
  return new Response(JSON.stringify(data), { 
    headers: { "Content-Type": "application/json" } 
  })
})
```

### Database Sync (Vector Store)

You can use Supabase logic to sync new database rows to Cencori Memory.

```sql
-- Example trigger to call Cencori API on new row
-- Note: Requires pg_net extension
create trigger on_new_document
  after insert on public.documents
  for each row execute function
  supabase_functions.http_request(
    'https://cencori.com/api/v1/memory',
    'POST',
    '{"content": "' || new.content || '"}'
  );
```
