# Cencori Company Bible

> Status: Living system  
> Purpose: Master index and table of contents for the Cencori Company Bible  
> Audience: Everyone at Cencori, present and future

---

## What This Is

The Cencori Company Bible is the internal operating reference for the company.

It is designed to ensure that:

- every team understands the same company
- every category fits into the same ecosystem
- every role can navigate directly to what matters to them
- every volume can cross-reference the others without creating conflicting truths

This is not a loose document set.
It is one company system with multiple role-specific volumes.

---

## How To Read This

Every employee should start with:

1. **Volume 0: Main Volume**
2. Their role-specific volume
3. Any referenced modules in the Canonical Models & References volume

The order matters.

The Main Volume defines the shared company truth.
The role-specific volumes define how each function should apply that truth.

---

## Cross-Reference Rule

All cross-volume references should use stable module IDs, not page numbers.

Example:

- `See ENG-07 for the technical model.`
- `See GTM-04 for the public narrative boundary.`
- `See REF-03 for the canonical primitive definition.`

This keeps the bible stable across revisions, redesigns, and future exports.
